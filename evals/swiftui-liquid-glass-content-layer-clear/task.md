# Dashboard Chart Surface Review

## Problem/Feature Description

A finance dashboard redesign places a large chart inside a translucent rounded panel over
a bright gradient background. The panel is purely visual; the only nearby action is a Play
button that opens a short market recap video. The team likes the extra depth but worries
the chart may become harder to read and that the effect may imply the whole card is a
control.

Review the design and give implementation guidance for the SwiftUI code path the team
should take on iOS 26 and earlier systems.

## Output Specification

Create a file named `swiftui-liquid-glass-content-layer-review.md` containing:

- Which element, if any, should receive Liquid Glass treatment.
- A SwiftUI snippet for the actionable element if relevant.
- Guidance for legibility over the bright background.
- Availability and fallback guidance.
