# SwiftUI Web Container Scope Review

## Problem/Feature Description

A team is building one SwiftUI app for iOS 26, iPadOS 26, macOS 26, and visionOS 26. It includes bundled help articles, a document-like in-app article reader with JavaScript table-of-contents support, GitHub OAuth, public legal and support links, and one legacy web surface that still depends on a custom `WKWebView` configuration.

## Output Specification

Write a concise implementation-routing review that chooses the right Apple web container for each part of the feature. Keep the answer practical and platform-aware.
