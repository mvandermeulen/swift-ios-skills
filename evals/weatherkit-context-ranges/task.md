# WeatherKit Context Queries And Ranges

## Problem/Feature Description

A product manager wants a WeatherKit feature that explains whether tomorrow's weather is unusual, can look back at forecast history, and avoids unsupported minute-by-minute precipitation forecasts. The team needs to know which WeatherKit queries to use and what range or availability constraints the implementation should enforce.

## Output Specification

Create `weatherkit-context-ranges.md` with the recommended WeatherKit queries, platform availability notes, date-range limits, and minute forecast safeguards. Include code snippets only where useful.
